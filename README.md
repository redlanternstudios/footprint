# Footprint

Footprint is a community opportunity network for LinkedIn and owned media.

Its purpose is to invite people to share what they are building, evaluate each submission fairly, and route every legitimate person toward a useful next outcome. Public features are one possible outcome. Clarification, encouragement, resources, introductions, discussion, and collaboration are equally valid outcomes.

Footprint helps Rory, Keymon, and RedLantern Studios through transparent participation in a community that creates value for others first.

Footprint is owned by By Red, LLC and built by RedLantern Studios.
